# Step 2/2 - 教学：第一次真正使用 Vim 编辑文件

## 任务速览
- 创建 `out/hello_vim.txt`
- 文件至少 2 行，并包含 `vim`

## 这一步你在学什么
现在开始，你要真正上手 Vim。

Vim 和很多图形编辑器最大的不同是：
**它不是“打开就直接输入”的编辑器。**

你至少需要知道两个模式：

### 普通模式
打开 vim 后默认就是普通模式。
这时键盘按键更多是在“发命令”。

### 插入模式
按 `i` 之后进入插入模式。
只有在这个模式里，你输入的字母才会真正写进文件。

## 本步任务
创建一个文件：
- `out/hello_vim.txt`

要求：
- 至少 2 行
- 文件里包含单词 `vim`

## 你会用到的命令 / 操作
### 1) `vim out/hello_vim.txt`
打开（或创建）文件。

### 2) `i`
进入插入模式。
你通常会在左下角看到 `-- INSERT --`。

### 3) `Esc`
从插入模式回到普通模式。

### 4) `:wq`
保存并退出。
- `w` = write
- `q` = quit

### 5) `:q!`
不保存强制退出。
如果你误操作了，这个命令很好用。

## 推荐你按这个顺序操作
1. 用 vim 打开目标文件
2. 按 `i`
3. 输入两行你自己的内容
4. 按 `Esc`
5. 输入 `:wq` 回车

## 你可以写什么
内容不需要复杂，例如：
- 第一行写你自己的话
- 第二行只要包含 `vim` 即可

重点不是“写什么”，而是你能否完成：
- 打开
- 编辑
- 保存
- 退出

## 自检
保存退出后运行：
```bash
cat out/hello_vim.txt
```
确认：
- 至少 2 行
- 包含 `vim`

完成后执行：
```bash
./quest submit
```
