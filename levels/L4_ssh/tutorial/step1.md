# Step 1/3 - 教学：生成一对 SSH 密钥

## 任务速览
- 在 `out/.ssh/` 生成一对 ed25519 密钥

## 这一步你在学什么
SSH 连接常见有两种方式：
- 输密码
- 用密钥

这一步只需要先理解最基本的一层：
- 私钥留在自己这里
- 公钥可以交给远端机器
- 远端用它确认你是否持有对应私钥

## 核心概念
### 1) `ssh-keygen` 是什么
`ssh-keygen` 是生成 SSH 密钥对的工具。

它不是登录命令，
也不是连机器的命令。
它只负责生成身份凭证。

### 2) 这题里几个参数分别是什么意思
这一步会用到 3 个关键参数：
- `-t ed25519`：指定密钥类型
- `-f <path>`：指定输出路径
- `-N ""`：空口令

你现在不需要记更多参数，
先把“生成什么、生成到哪里”这件事搞明白就够了。

### 3) 为什么生成到 `out/.ssh/`
这一步不是在改你机器上真实的 `~/.ssh/`。

题目把产物放在 `out/.ssh/`，
是为了：
- 让练习范围只留在当前关卡里
- 方便验收
- 方便你回看这一步做了什么
