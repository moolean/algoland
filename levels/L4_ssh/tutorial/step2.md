# Step 2/3 - 教学：给 SSH 连接写一个可复用别名

## 任务速览
- 创建 `out/.ssh/config`
- 写入别名 `train-node` 及对应连接信息

## 这一步你在学什么
如果每次 SSH 都手打一长串参数，
很容易出错，也不方便记。

SSH config 的作用很简单：
**把一组连接参数收成一个名字。**

以后你记住这个名字就行。

## 核心概念
### 1) `Host` 是什么
`Host` 是你自己定义的别名，
不是远端机器的真实地址。

例如这题里的：
- `train-node`

它的作用是告诉 SSH：
“当我用这个名字时，请套用下面这组配置。”

### 2) 这一步里最关键的 4 个字段
- `HostName`：真实主机地址
- `User`：登录用户名
- `Port`：连接端口
- `IdentityFile`：使用哪把私钥

你不需要一次记很多 SSH 配置项，
先掌握这 4 个最常见的就够了。

### 3) 这一步最容易错的地方
常见错误包括：
- 把 `Host` 当成地址来写
- `IdentityFile` 路径写错
- 字段名拼错
- 别名不是题目要求的名字
